Easily add a UK library finder as a widget to your site.

== Description ==

The WordPress widget gives WordPress site owners the option to embed a library finder as part of their site.

This is an unbranded, easily customisable widget allowing admins the chance to enter their own title and description text, and search for either public or non-public libraries (or both).

Optimised for UK libraries.

== Installation ==

1. Upload `library-finder-widget.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

The widget will then be available to use.
